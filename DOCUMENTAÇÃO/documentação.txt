Clip CLI = Agrupador ou Gereciador de Console 

#requisito de usuario
desenvolver uma aplicação capaz de gerenciar a saida de console de varios tipos de services. 
Exemplo: em meu projeto pessoal tenho uma aplicação sendo desenvolvida utilizando reactjs, sass do lado client, e Nodejs do lado servidor.
Sempre que preciso iniciar meu ambiente de desenvolvimento, preciso iniciar o server, colocar a compilação do sass em esculta, e iniciar o server do react para emular a aplicação do Client. 

#requisito de sistema
Para que o software seja flexivel será necessario que o usuario configure de acordo com os requisitos do sistema em questão, ou seja,
será preciso obter do usuario quais são os comandos ele deseja roda e em qual diretorio. 

#requisito não funcional
O software será desenvolvido em nodejs, é será distribuido pelo NPM como um nome module(CLI) global.
