Crie ambientes de desenvolvimento com o CLIP, e inicie tudo muito rápido!
Instale com

$ npm install -g clip

Inicie o React-Native , Sass ou Docker
e qualquer outro serviço necessário no ambiente de desenvolvimento do seu projeto.
COMO USAR:
Para criar um novo ambiente ou adicionar comandos ao ambiente:

Clip config "Nome do Projeto"" "Diretorio do projeto" "Comando"
Exemplo:
clip config AppMobile /home/Documentos/clients/com.stream.app npm start


Para iniciar um ambiente:

Clip run "Nome do Projeto"
Exemplo:
clip run AppMobile


Quanto custa?

A licença do Clip custa apenas R$ 2, um valor simbólico, para nos ajudar no desenvolvimento do projeto.
